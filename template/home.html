Smart Attendance System

Maintaining the attendance is very important in all the organizations for checking the performance of an Employee/Student. Every organization has its own method in this regard. Some are marking attendance manually using the old paper or file based approach and some have adopted methods of automatic attendance using some biometric techniques. But in these methods employees/students have to wait for long time in making a queue at time they enter the office/school. In this project, we introduced Smart attendance system based on face recognition that can be used for taking attendance of the employees/students entering in to the office/school.
